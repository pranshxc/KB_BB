Shell Name: Mysql interface v1.0.php  
URL: <https://www.antihack.me/public/kyc/webshell/php/PHPshell/Mysql%20interface%20v1.0/Mysql%20interface%20v1.0.php>

So,** it's possible to visit URL using intercept the request, Then i intercept the request and i can able to view the shell page then i report to [**Anti Hack.me**](https://antihack.me/) They offer me Gold Coin And SWAG.
