# Facebook employee internal tool and conversations leaked in Facebook video

There is a Facebook page that Facebook employees and Facebook contractors use for checking new features through Facebook. An employee seems to have left **a 7-minute Facebook live video** with their internal Facebook activity. The activity listed messenger conversations between many employees, Google presentations for upcoming Facebook page changes, internal tools such as FBQuip, EntTool GateKeeper and Facebook employee profile on-call schedules.  
  
The page with the live video was named  
https://www.facebook.com/**BestTestPageEver** /  
  
**Timeline**  
  
Sep 12, 2019 8:14 PM – Report sent (Video posted an hour or two earlier)  
Sep 12, 2019 8:28 PM – Confirmation by Facebook (Page taken down)  
Sep 21, 2019 – Bounty awarded by Facebook
