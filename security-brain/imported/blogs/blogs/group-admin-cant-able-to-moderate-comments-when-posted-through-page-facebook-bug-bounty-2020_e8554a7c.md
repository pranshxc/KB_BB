Hello Everyone, I am Prakash Panta From Bharatpur Nepal. Today i will tell you about my 2nd valid bug on Facebook. Let’s get started.

As we know that Facebook Groups Can Linked the Facebook Pages. In my case when linked pages posted something in Facebook Groups and someone’s comments in that post, In such case Group Admin is not able to moderate comments using FBLite.

Once reported, they acknowledged my report,they closed the report as informative,they find more impact due to my report,they fixed the issue and finally rewarded me with bounty.
