# Group Admin Can’t Able to Moderate Comments When Posted Through Page : Facebook Bug Bounty 2020

__Dec 30, 2020

## Introduction

Hello Everyone, I am Prakash Panta From Bharatpur Nepal. Today i will tell you about my 2nd valid bug on Facebook. Let’s get started.

## Explanation of Issue

As we know that Facebook Groups Can Linked the Facebook Pages. In my case when linked pages posted something in Facebook Groups and someone’s comments in that post, In such case Group Admin is not able to moderate comments using FBLite.  
Once reported, they acknowledged my report,they closed the report as informative,they find more impact due to my report,they fixed the issue and finally rewarded me with bounty.  

![Message From Facebook Security Team](/posts/images/2/2-1.jpg)

![Message From Facebook Security Team](/posts/images/2/2-2.jpg)

## Timeline of Report

June 13, 2020: Initial Report Sent

June 17, 2020: Not Reproduced From Facebook Security Team and Further Info Sent.

June 19, 2020: Still Not Reproduced and More details sent.

June 23, 2020: Pre-Triaged

July 8, 2020: Triaged

July 16, 2020: Closed as Informative due to low impact (I didn’t review requested/Reopen the report)

August 11, 2020: Fixed and Bounty Rewarded (They Re-open Report and Provide me Bounty)
